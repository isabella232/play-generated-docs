Related Docs: object ServerConfig | package server

case class ServerConfig(rootDir: File, port: Option[Int], sslPort: Option[Int], address: String, mode: api.Mode.Mode, properties: Properties, configuration: api.Configuration) extends Product with Serializable

Common configuration for servers such as NettyServer.

rootDir

The root directory of the server. Used to find default locations of files, log directories, etc.

port

The HTTP port to use.

sslPort

The HTTPS port to use.

address

The socket address to bind to.

mode

The run mode: dev, test or prod.
